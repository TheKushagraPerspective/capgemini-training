# CSS Flexbox and Grid (Detailed Explanation)

---

# 1. Flexbox (1-D Layout)

Flexbox is used to arrange elements in one direction:

* Row (horizontal)
* Column (vertical)

---

## 2. Flex Container (Parent Properties)

### display: flex

```css id="f1"
display: flex;
```

* Activates flexbox
* All direct children become flex items
* Default direction becomes row

---

### flex-direction

```css id="f2"
flex-direction: row;
```

Defines the main axis (direction of layout)

Values:

* row → left to right (default)
* column → top to bottom
* row-reverse → right to left
* column-reverse → bottom to top

Important:
Main axis changes based on direction

---

### justify-content

```css id="f3"
justify-content: center;
```

Controls alignment along the main axis

How it works:

* Moves items horizontally (if row)
* Moves items vertically (if column)

Values:

* flex-start → start of container
* flex-end → end
* center → center
* space-between → equal space between items
* space-around → equal space around items
* space-evenly → equal spacing everywhere

---

### align-items

```css id="f4"
align-items: center;
```

Controls alignment along the cross axis

How it works:

* If flex-direction = row → vertical alignment
* If flex-direction = column → horizontal alignment

Values:

* flex-start
* flex-end
* center
* stretch (default)
* baseline

---

### flex-wrap

```css id="f5"
flex-wrap: wrap;
```

Controls whether items move to next line

Values:

* nowrap → all items stay in one line
* wrap → items move to next line
* wrap-reverse → reverse wrapping

---

### align-content

```css id="f6"
align-content: center;
```

* Works only when multiple rows exist (flex-wrap required)
* Aligns rows (not items)

---

### gap

```css id="f7"
gap: 10px;
```

* Adds spacing between items
* Cleaner than margin

---

## 3. Flex Items (Child Properties)

---

### flex-basis

```css id="f8"
flex-basis: 100px;
```

* Defines initial size of item
* Works like width (in row) or height (in column)

---

### flex-grow

```css id="f9"
flex-grow: 1;
```

* Controls how much item grows
* Extra space is distributed based on value

Example:

* Item1 → flex-grow: 1
* Item2 → flex-grow: 2
  → Item2 gets double space

---

### flex-shrink

```css id="f10"
flex-shrink: 1;
```

* Controls shrinking when space is less
* Higher value shrinks more

---

### order

```css id="f11"
order: 2;
```

* Changes visual order of items
* Default = 0

Lower value appears first

---

### align-self

```css id="f12"
align-self: center;
```

* Overrides align-items for single item

---

### flex (shorthand)

```css id="f13"
flex: 1 1 100px;
```

* Combines:

  * flex-grow
  * flex-shrink
  * flex-basis

---

# 4. Grid (2-D Layout)

Grid is used for both rows and columns together.

---

## 5. Grid Container (Parent Properties)

### display: grid

```css id="g1"
display: grid;
```

Enables grid layout

---

### grid-template-columns

```css id="g2"
grid-template-columns: 1fr 1fr 1fr;
```

* Defines columns
* 1fr = 1 fraction of available space

Example:
3 equal columns

---

### grid-template-rows

```css id="g3"
grid-template-rows: auto auto;
```

Defines rows

---

### gap

```css id="g4"
gap: 10px;
```

Space between rows and columns

---

### justify-content

* Aligns entire grid horizontally

---

### align-content

* Aligns entire grid vertically

---

### place-content

```css id="g5"
place-content: center;
```

Shorthand for:

* justify-content
* align-content

---

## 6. Grid Items (Child Properties)

---

### grid-column-start / grid-column-end

```css id="g6"
grid-column-start: 1;
grid-column-end: 3;
```

* Item spans from column 1 to 3

---

### grid-row-start / grid-row-end

```css id="g7"
grid-row-start: 1;
grid-row-end: 3;
```

* Item spans rows

---

### grid-column (shorthand)

```css id="g8"
grid-column: 1 / 3;
```

---

### grid-row (shorthand)

```css id="g9"
grid-row: 1 / 3;
```

---

### place-self

```css id="g10"
place-self: center;
```

* Aligns single grid item

---

# 7. Important Concepts

## Main Axis vs Cross Axis (Flexbox)

* Main axis → defined by flex-direction
* Cross axis → perpendicular to main axis

---

## Flexbox vs Grid

| Feature  | Flexbox   | Grid         |
| -------- | --------- | ------------ |
| Layout   | 1D        | 2D           |
| Control  | Limited   | Powerful     |
| Use case | Alignment | Full layouts |

---

# 8. When to Use

Use Flexbox:

* Navbar
* Buttons alignment
* Small layouts

Use Grid:

* Page layout
* Complex UI
* Dashboard

---

# 9. Summary

* Flexbox handles one direction layout
* Grid handles two-dimensional layout
* justify-content → main axis
* align-items → cross axis
* flex-grow → controls expansion
* grid-template → defines structure
