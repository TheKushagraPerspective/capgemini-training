DOM (Document Object Model)

indirect access :
1. getElementById("id_name") :- if you pass a invalid id_name that is not exist then it will return null

2. getElementByClassName("class_name") :- if you pass a invalid class_name that is not exist then it will return empty HTML Collection (just like an array but not the array, and to use array function, first convert HtmlCollection into array (Array.from()) then use it)

3. getElementByTagName("tag_name") :- if you pass a invalid tag_name that is not exist then it will return empty HTML Collection (just like an array but not the array, and to use array function, first convert HtmlCollection into array (Array.from()) then use it)

4. getQuerySelector("selector") :- if you pass a invalid selector that is not exist then it will return null (if selector is id then pass :- "# + id_name" , is selector is class then ". + class_name", if selectod is tag then simply pass "tag_name");

5. getQuerySelectorAll("selector") :- if you pass a invalid selector that is not exist then it will return empty NodeList (just like an array but not the array, and we can directly use array function like forEach(), no need to convert it into array first)
