# JavaScript Functions

## 1. What is a Function

A function is a block of code that performs a specific task and can be reused.

---

## 2. Types of Functions

### 1. Named Function

A function with a name.

```js id="f1"
function greet() {
    console.log("Hello");
}
```

---

### 2. Anonymous Function

A function without a name.

```js id="f2"
let greet = function() {
    console.log("Hello");
};
```

---

### 3. Function Expression

When a function is stored inside a variable.

```js id="f3"
let sum = function(a, b) {
    return a + b;
};
```

---

### 4. Nested Function

A function inside another function.

```js id="f4"
function outer() {
    let message = "Hello";

    function inner() {
        console.log(message);
    }

    inner();
}
```

---

### 5. Callback Function

A function passed as an argument to another function.

```js id="f5"
function greet(name, callback) {
    console.log("Hello " + name);
    callback();
}

function sayBye() {
    console.log("Bye");
}

greet("Kushagra", sayBye);
```

---

### 6. Higher Order Function

A function that:

* accepts another function as argument
* or returns another function

```js id="f6"
function operation(a, b, callback) {
    return callback(a, b);
}

function add(x, y) {
    return x + y;
}

operation(2, 3, add);
```

---

### 7. Arrow Function

Short syntax for writing functions.

```js id="f7"
let sum = (a, b) => a + b;
```

---

## 3. Closure

Closure is a concept where an inner function can access variables of its outer function even after the outer function has finished execution.

It works because JavaScript creates a special memory (closure object) that stores outer function variables.

### Example

```js id="f8"
function outer() {
    let count = 0;

    return function inner() {
        count++;
        console.log(count);
    };
}

let counter = outer();
counter(); // 1
counter(); // 2
```

---

## 4. Important Concepts

### Function Parameters vs Arguments

```js id="f9"
function add(a, b) {   // parameters
    return a + b;
}

add(2, 3);             // arguments
```

---

### Default Parameters

```js id="f10"
function greet(name = "Guest") {
    console.log(name);
}
```

---

### Rest Parameters

```js id="f11"
function sum(...nums) {
    return nums.reduce((acc, val) => acc + val, 0);
}
```

---

### Return Keyword

* Returns value from function
* Stops execution after return

```js id="f12"
function square(n) {
    return n * n;
}
```

---

## 5. Differences

### Callback vs Higher Order Function

* Callback → function passed as argument
* Higher Order → function that accepts or returns function

---

### Normal Function vs Arrow Function

| Feature      | Normal Function | Arrow Function       |
| ------------ | --------------- | -------------------- |
| Syntax       | Longer          | Short                |
| this keyword | Own this        | Inherits from parent |
| Use case     | General purpose | Short functions      |

---

## 6. Common Mistakes

* Forgetting return keyword
* Confusing function declaration and expression
* Using arrow function where this is required
* Not understanding closure behavior

---

## 7. Summary

* Functions are reusable blocks of code
* Types include named, anonymous, arrow, callback, higher order
* Closures allow access to outer variables
* Higher order functions improve reusability
* Arrow functions provide shorter syntax
