# TypeScript Notes

## 1. What is TypeScript

TypeScript is a superset of JavaScript that adds static typing.
It helps in writing safer and more maintainable code.

---

## 2. Types in TypeScript

### Static Types

TypeScript provides static typing, which means types are checked at compile time.

#### Built-in Types (Primitive)

* string
* number
* boolean
* null
* undefined

#### User Defined Types

* Array
* Object
* Tuple
* Interface
* Type (Type Alias)

#### Special Types

* any → disables type checking
* unknown → safer alternative to any
* void → used for functions that do not return anything
* never → used for functions that never complete (like throwing errors or infinite loops)

---

## 3. Type Inference

TypeScript can automatically detect the type of a variable based on the value assigned.

### Example

```ts id="a1b2c3"
let name = "Kushagra";   // inferred as string
let age = 22;            // inferred as number
```

### Key Points

* No need to explicitly define types
* Useful for small scale projects
* Recommended mainly for primitive types

---

## 4. Type Annotation (Explicit Typing)

When we manually define the type of a variable.

```ts id="d4e5f6"
let name: string = "Kushagra";
let age: number = 22;
```

---

## 5. Narrowing

Narrowing means converting a general type into a more specific type.

### Example

```ts id="g7h8i9"
function printValue(value: string | number) {
    if (typeof value === "string") {
        console.log(value.toUpperCase()); // string
    } else {
        console.log(value.toFixed(2));    // number
    }
}
```

Here, TypeScript narrows the type based on condition.

---

## 6. Generics

Generics allow us to write reusable and flexible code.

### Example

```ts id="j1k2l3"
function identity<T>(value: T): T {
    return value;
}

identity<number>(10);
identity<string>("hello");
```

---

## 7. Interfaces

Used to define structure of an object.

```ts id="m4n5o6"
interface User {
    name: string;
    age: number;
}

let user: User = {
    name: "Kushagra",
    age: 22
};
```

---

## 8. Type Alias

Alternative way to define types.

```ts id="p7q8r9"
type User = {
    name: string;
    age: number;
};
```

---

## 9. Tuples

Used when we know the exact number and type of elements.

```ts id="s1t2u3"
let user: [string, number] = ["Kushagra", 22];
```

---

## 10. Decorators (Advanced Concept)

Decorators are special functions used to modify classes or class members.
Commonly used in frameworks like Angular.

```ts id="v4w5x6"
function Log(target: any) {
    console.log("Class created");
}

@Log
class Demo {}
```

---

## 11. Summary

* TypeScript adds static typing to JavaScript
* Type inference automatically detects types
* Narrowing helps make types more specific
* Generics provide reusable code
* Interfaces and Types define structure
* Special types handle edge cases
* Decorators are used in advanced frameworks like Angular
